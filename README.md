## 📌 주요 기능

🍽️ **주문자 모드**  
- 위치와 카테고리 별 국민대 내 매장 검색
- 픽업/전달 중 선택하여 메뉴 미리 주문
- 주문 내역 및 현재 주문 진행 상황 확인 가능

📦 **전달자 모드**  
- 픽업 매장 위치별 필터링
- 전달 내역 및 현재까지의 수입 확인 가능 

---

## 🌟 서비스 포인트

1️⃣ **이제 휴대폰으로 간편하게 픽업 주문이 가능해요!**
- 수업 때문에 시간에 쫓겨가며 기다릴 일이 없어요.
- 특정 메뉴를 먹기 위해 멀리 이동할 필요 없어요.

2️⃣ **각 매장의 메뉴를 미리 볼 수 있어요**

3️⃣ **전달 기능을 통해 300원이라는 적은 비용으로 음식을 받을 수 있어요**

4️⃣ **직접 전달자가 되어 여유 시간에 수익을 얻을 수 있어요**
